# Only Post - Twitter(X) Focus Extension

[English](#english) | [日本語](#japanese)

<a name="english"></a>

## English

A Chrome extension to prevent overusing Twitter (X) by limiting access to only essential information (your profile and posting).

### Features

- **Profile-only Access**: Your profile (`x.com/username`) is always accessible.
- **10 Daily Stocks**: Viewing other pages (Home, Search, etc.) consumes one "stock".
- **1-Minute Timer**: Each stock gives you 1 minute of browsing time.
- **Safe Posting**: The timer **pauses** while you are on the "Compose Post" screen or typing.
- **Auto Block**: After 1 minute, the screen is blocked to encourage returning to your profile.

### Installation

1. Download and extract this folder.
2. Open `chrome://extensions/` in Chrome.
3. Enable "Developer mode" (top right).
4. Click "Load unpacked" and select this folder.

## 日本語

Twitter（X）の使いすぎを防ぎ、必要な情報（自分のプロフィールや投稿）だけに集中するためのChrome拡張機能です。

### 主な機能

- **プロフィール限定アクセス**: 自分のプロフィール画面は制限なく閲覧可能です。
- **1日10回のストック制**: それ以外の画面を見るには「ストック」を1つ消費します。
- **1分間のタイマー**: ストック1つにつき、1分間だけ閲覧できます。
- **投稿画面の保護**: ポスト作成中や投稿画面では**タイマーが停止**します。
- **自動ブロック**: 1分経過すると画面がブロックされ、プロフィールへの戻りを促します。

### インストール方法

1. このフォルダをダウンロードして展開します。
2. Chromeで `chrome://extensions/` を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」から、このフォルダを選択します。
